Java é uma programação 100% orientada a objeto.

Pilares da orientação a objeto:
    - Encapsulamento
    - Herança
    - Polimorfismo


    - Singleton (Instância única)
    - Public    (todos os lugares)
    - Private
    - Protected   (mesmo pacote)

Organizando o código
    Pacotes:
        - Sempre em minúsculo Ex: br.com.camile.pacote
        - Servem para organizar o código fonte
        - nomes de pacotes NÂO podem ter:
            - letras maiúsculas, espaço, sem caracteres especiais, acentos;
        Convenção:
            - domnínio da empresa
            - último nome é o específico do pacote

Getters e Setters
    Getters
        - É um método público para consultar dados;
            - Ele só retorna o valor do atributo;
        - nomenclatura:
            get_nomeAtributo;        

        //método getter
        Exemplo: sessao11
            public float getSaldo(){
                return this.saldo;
            }

Public:
    - Acesso em todo projeto, podemos importar para outros pacotes

Private:
    - Só pode ser acessado pela classe

Protected:
    - Padrão do java.
    - Acesso no mesmo pacote em que está.
